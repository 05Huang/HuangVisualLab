# ✨ HuangVisualLab

> 🚀 Creative Coding / Visual Effects / Interactive Experiments

**HuangVisualLab** 是一个专注于视觉特效与创意编程的实验仓库，收集并实现各类基于 **HTML / CSS / JavaScript / Python** 的动画与交互效果。

旨在通过代码探索视觉表达、用户交互与算法之间的结合。

## 🧪 Features（内容方向）

- 🎨 前端动画（CSS / Canvas / WebGL）
- 🌀 粒子系统 & 物理模拟
- 🤖 AI + 视觉交互（MediaPipe / 手势识别）
- 🖼️ Python 图像处理（OpenCV / Pillow）
- ✨ Creative Coding（创意编码）


## 🚀 Getting Started

### ▶️ HTML Effects

直接打开 `.html` 文件即可运行：

```bash
open index.html
```

------

### ▶️ Python Effects

安装依赖后运行：

```bash
pip install -r requirements.txt
python demo.py
```
